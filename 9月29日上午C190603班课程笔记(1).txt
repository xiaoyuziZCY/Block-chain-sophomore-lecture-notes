4、Go语言环境安装：

		GOROOT：该变量的值是Go语言软件的安装目录，一般通常为C:Go

8、Go语言的基本语法：
变量：本质是一种代称，可以根据需求指定具体所代表的内容
① var 变量名 数据类型
② 变量名 := 数据值
③ 变量名1, 变量名2, 变量名3 := 数值1，数值2，数值3

数据类型：不同类型的数据，对数据进行分类。比如说人：男人和女人。

Go语言的基本数据类型：
	string：字符串，以双引号包裹,比如 "hello world"
	int: 数值类型中的整形, 既可以表示负数，也可以表示整数比如1，2。int8、int16、int32、int64等，表示的数据的范围不同
		int8表示的范围：[-128,127]
	unint：数值类型中的整形, 无符号整形, 所表示的数据全是正整数,uint8、uint16、uint32..
		uint8表示的范围：[0,255]

	float：浮点数,带小数点。float32和float64,区别是范围不同
	bool：布尔类型,取值有两种：true、false

格式化打印：通过程序向控制台输出信息，称为打印。
控制台：CMD、浏览器开发者工具console、goland调试区等均属于控制台
	输出：需要用到fmt包，使用import进行导入
		fmt.Print()：输出内容，没有换行。 可以使用\n进行换行,\n称之为转义字符
		fmt.Println()：输出内容，并输出一个换行
		fmt.Printf(格式,数据内容)：
			%s：字符串
			%d：整数
			%x：hex十六进制。
			%p：变量的地址，pointer

数据类型的转换：
	整数转字符串：strconv.Itoa, 返回值类型为string
	字符串转整数：strconv.Atoi, 返回值类型是int和error

常量：数据内容永远不会变化的称之为常量
声明方式：const 常量名 = 数据内容值
规范：常量名一般使用全部大写的方式进行声明
特殊的常量：iota

  		  } else{

  		  }； 
  		  if 条件表达式1 {

  		  } else if 表达式2{

  		  }else 等
  		  swtich .... case
  循环：

  			//循环体程序
  		}

  		③ goto：跳转到程序指定行执行。

11、Go语言的函数和指针
函数：是可以完成某种特定功能的代码块，可以被多次调用执行。与方法不同，不需要接收值类型。
函数的声明格式：关键字func表示函数
func 函数名(参数列表) (返回值类型列表){
	return 返回值列表
}
函数的使用：
	① 函数的声明和定义
	② 在其他需要的地方调用
参数的类型：
	① 形式参数：函数在声明的时候，定义的用于接收的参数，称之为形式参数。该参数没有实际含义，只是一个代称。
	② 实际参数：在调用函数的时候，给该函数传递的参数，其中包含具体的数据内容，此时传递的参数称之为实际参数，简称为实参。

12、数据容器：存放数据的结构称之为容器
	① 数组： 存放相同类型的长度固定的一组数据的结构，称之为数组。在内存中，数组的分配是连续的。
		使用方法：[size]int
	② 切片：存放相同类型的长度不固定可变的一组数据的机构，称之为切片。使用场景更多
		使用方法：[]int，make([]int,0)
		容量：cap(),
		长度：len()
	③ Map：地图，映射。用于描述一组数据之间的映射关系，以key为键，以value为值。更具key可以获取对应的映射到的值value，其中key是唯一的，不能重复。这种数据结构称之为map.
		使用方法：
			map1 := make(map[string]int)
